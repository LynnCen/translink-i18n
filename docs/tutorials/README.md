# TransLink I18n 技术教程

本教程系列深入讲解 TransLink I18n 的技术实现，帮助你理解如何从零构建一个现代化的国际化解决方案。

1. [Monorepo 架构设计](./01-monorepo-architecture.md)
   - pnpm Workspace 配置
   - Turborepo 构建优化
   - TypeScript 项目引用
   - 包依赖管理

2. [CLI 工具开发](./02-cli-development.md)
   - Commander.js 命令系统
   - AST 文本提取
   - 哈希生成算法
   - Excel/CSV 导入导出

3. [Runtime 运行时实现](./03-runtime-implementation.md)
   - 翻译引擎核心
   - 缓存管理策略
   - 事件系统设计
   - 框架适配器（Vue/React）

### 进阶篇

4. [Vite 插件开发](./04-vite-plugin.md)
   - Vite 插件机制
   - 虚拟模块系统
   - HMR 热更新
   - 代码转换

5. [插件系统设计](./05-plugin-system.md)
   - 插件接口设计
   - 生命周期管理
   - 插件加载机制
   - Vika 插件实现

6. [构建与优化](./06-build-optimization.md)
   - tsup 构建配置
   - Tree-shaking 优化
   - 类型声明生成
   - 包大小优化

7. [AI 翻译功能实现](./07-ai-translation.md) 🆕
   - AI Provider 抽象层设计
   - 多AI模型集成
   - 批量翻译优化（降低95%成本）
   - 智能缓存系统
   - 错误处理与重试机制
